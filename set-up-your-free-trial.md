# Set Up Your Free Trial

Create your free Grafana Cloud account [here](https://grafana.com/auth/sign-up?refCode=gr8PfdbwkzKhg9R).

1.  Choose your preferred login method.  Account creation starts a 14-day trial period that allows you to send unlimited data and create unlimited users to test Grafana Cloud at scale. When the trial ends, you transition to the free-forever plan where you can do quite a bit with it!

2. Choose your deployment region when setting up your first Cloud stack.

   <img width="375" height="332" alt="image" src="https://github.com/user-attachments/assets/1047335f-0169-4dfa-821a-df7eee76a3db" />

3. You are all set to start exploring Grafana Cloud!
